We have a Bejeweled style game board. We want to find out how many switches will clear cells.
A game board is a 2-dim list containing some elements.
A switch is switching two adjacent elements in board.
A switch clear cells if there is at least three adjacent cells contain same elements in alignment after switching.

Here is an example:

[7,3,1,4,4]
[7,3,4,3,2]
[3,7,5,1,2]
Result: 2
After switching 3 and 7, we got three 7 and three 3 in alignment, so the switch clears cells.

Input: 2-dim list, the state of a game board. Different numbers mean different kinds of element, there are at most 5 kinds of elements

Output: Integer, how many switches can clear cells.

Example: 

    assert checkio(
    [[2,1,3,2],
     [1,1,3,3],
     [2,2,1,3],
     [1,1,3,2],
     [3,2,3,2]]) == 7
